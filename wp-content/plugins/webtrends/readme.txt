A plugin to integrate Webtrends with your WordPress blog. Provides an Analytics dashboard as well.

== Description ==

Webtrends founded the web analytics industry in 1993. Today, their leadership extends to measurement, optimization and integration of all digital content, including web sites, social media and paid-search advertising.

This Webtrends WordPress plugin allows Webtrends customers to:
*  Automatically add the necessary JavaScript to their WordPress footer for tracking
*  With an API Key entered, allows users to view their Analytics data in a WordPress dashboard.

== Installation ==

1. You MUST have an active Webtrends account in order to use the plugin
1. Upload the `webtrends` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to `Settings/Webtrends` and enter your Webtrends credentials

== Frequently Asked Questions ==

= Does this plugin allow access to all my Webtrends reporting =

No.  There's no substitute for the incredible reporting in our interface.  We just wanted to provide WordPress users with a robust chart and some basic data that will help their users monitor their blog traffic.
